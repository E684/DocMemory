# 感想
全体を通してのキーワードを選ぶとすれば、「ソース」と「クリエイティブフィールド」
- ソース
  - タイトルにある「1人」が指す存在。ビジョンの実現に最も深くかかわる存在だが、役職などの面に関わらず意識面での関係を重視している。
- クリエイティブフィールド
  - 人と組織が動き出す「創造的な活動の場」で、ソースを中心として活動に関わる人を俯瞰的に見るための方法。やはり組織構造にとらわれないで、各々がどのような動機や期待をもって関わっているかを基準とする。
